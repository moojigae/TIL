폼관련 태그

form태그는 html에서 사용자가 입력할 수 있는 양식 제공
form태그 그 내의 input태그들을 통해 사용자가 입력한 정보를 서버로 넘김
* action 속성: 폼에 입력된 값들을 전송받을 서버의 클래스 명 입력 * method 속성: get/post방식으로 전송 방식 지정. 기본 설정은 get방식

검색할 내용 :

submit버튼을 눌렀을 때 method에 지정한 방식으로 action에 지정한 경로에 input태그의 값을 전달함 input type="submit"을 넣으면 버튼에 기본 제출이 들어옴

필드셋의 제목을 작성하는 부분



필드셋으로 묶은 영역별로 구분됨


text와 관련된 input태그 속성들

type="hidden

- 사용자 눈에는 보이지 않지만 값을 담고 보내고 할 수 있음

type="text"

- 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 상자


type="password"

- 비밀번호를 입력할 수 있는 텍스트 상자

비밀번호 :

type="button", type="submit", type="reset"

button태그를 이용해서도 버튼을 만들 수 있음
