배경 스타일

background-color

배경 색을 지정할 때 사용

	- 선택자{background-color: 색상;}

background-clip

배경 범위 조절할 때 사용

	- 선택자{background-clip: 속성 값;}

배경 이미지

background-image

배경에 이미지를 넣고 싶을 대 사용

	- 선택자{background-image: url(경로);}

background-repeat

배경에 이미지를 반복해서 넣고 싶을 때 사용

	- 선택자{background-repeat: 속성 값;}

background-size

배경 이미지 크기를 조절하고 싶을 때 사용

	- 선택자{background-size: 속성 값;}

background-position

배경 이미지 위치를 조정하고 싶을 때 사용

	- 선택자{background-position: 속성 값;}

background-origin

배경 이미지 배치 시 기준을 정하고 싶을 때 사용

	- 선택자{background-origin: 속성 값;}

background-attachment

스크롤이 움직임에 따라 배경이 움직일지 움직이지 않을지 정할 때 사용

	- 선택자{background-attachment: scroll | fixed;}
